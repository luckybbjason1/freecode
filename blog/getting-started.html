Claude Code 한국어 시작 가이드

Claude Code를 처음 사용하는 분들을 위한 완전한 한국어 가이드. 설치부터 첫 번째 에이전트 실행까지 단계별 안내.

Claude Korea·2025년 1월
핵심 요약 (TL;DR)
  • Claude Code는 Anthropic이 만든 터미널 기반 에이전트형 AI 코딩 도구로, 파일 편집·명령어 실행·엔드투엔드 작업을 수행한다.
  • 설치 요구사항은 Node.js 18+와 console.anthropic.com에서 발급한 Anthropic API 키이며 Windows·macOS·Linux를 모두 지원한다.
  • 설치는 npm install -g @anthropic-ai/claude-code, 실행은 프로젝트 폴더에서 claude 명령어로 한다.
  • 자연어로 작업을 지시하며, 스킬은 claude skill install tdd-workflow 형태로 설치한다.
  • 기본 명령어는 /help, /clear, /commit, /review, /plan 등이다.

Claude Code란 무엇인가?

Claude Code는 Anthropic이 개발한 터미널 기반 AI 코딩 어시스턴트입니다. 단순한 코드 생성 도구를 넘어, 파일을 읽고 편집하며, 명령어를 실행하고, 복잡한 소프트웨어 엔지니어링 작업을 처음부터 끝까지 수행할 수 있는 강력한 에이전트형 도구입니다.

Claude Code는 여러분의 개발 워크플로우를 자동화하는 가장 강력한 AI 도구입니다. 에이전트, 스킬, 훅을 통해 무한히 확장 가능합니다.

설치 요구사항

  • Node.js 18+ — 공식 사이트에서 다운로드
  • Anthropic API 키 — console.anthropic.com에서 발급
  • 운영체제 — Windows, macOS, Linux 모두 지원

Claude Code 설치하기

터미널을 열고 다음 명령어를 실행합니다:

npm install -g @anthropic-ai/claude-code

설치가 완료되면 API 키를 설정합니다:

export ANTHROPIC_API_KEY="your-api-key-here"

첫 번째 실행

프로젝트 디렉토리에서 claude 명령어를 실행합니다:

cd your-project
claude

Claude Code가 실행되면 자연어로 작업을 지시할 수 있습니다. 예를 들어 "이 파일에 있는 버그를 찾아서 수정해 줘"와 같이 입력하면 됩니다.

스킬 설치하기

Claude Code의 진정한 힘은 스킬과 에이전트에 있습니다. 이 마켓플레이스에서 원하는 스킬을 찾아 설치하세요:

# 스킬 ID를 복사하여 설치
claude skill install tdd-workflow

기본 명령어

  • /help — 도움말 보기
  • /clear — 대화 기록 초기화
  • /commit — 자동 커밋 메시지 생성
  • /review — 코드 리뷰 실행
  • /plan — 구현 계획 작성
